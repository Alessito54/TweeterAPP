🚀 SIGUIENTE: PASOS A SEGUIR
═══════════════════════════════════════════════════════════════════════════════

1️⃣  CONFIGURAR BASE DE DATOS
   └─ Ir a QUICK_START.txt (sección Paso 1.1)
   └─ Crear cuenta Aiven.io → PostgreSQL 17
   └─ Copiar credenciales
   └─ Editar: tweeter-api/src/main/resources/application.properties
      
2️⃣  COMPILAR BACKEND
   └─ cd tweeter-api
   └─ mvn clean package
   └─ Verificar: target/tweeter-api-0.0.1-SNAPSHOT.jar ✓

3️⃣  DOCKERIZAR
   └─ docker build -t tu-usuario/tweeter-api:latest .
   └─ docker push tu-usuario/tweeter-api:latest

4️⃣  DESPLEGAR EN RENDER
   └─ Ir a render.com
   └─ New Web Service (container image)
   └─ Pegar: tu-usuario/tweeter-api:latest
   └─ Agregar variables de ambiente (Aiven credentials)
   └─ Deploy → Copiar URL pública

5️⃣  CONFIGURAR FLUTTER
   └─ cd flutter_client
   └─ flutter pub get
   └─ Editar: lib/screens/home_screen.dart
   └─ Reemplazar customBaseUrl: tu URL de Render
   └─ flutter run

📞 SOPORTE & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problema: "Connection refused"
  ├─ ¿Backend corriendo? java -jar target/*.jar
  └─ ¿URL correcta? Usar 10.0.2.2 en emulador Android
  
Problema: "SSL certificate error"
  ├─ Verificar sslmode=require en URL
  └─ En dev, cambiar a http, no https
  
Problema: "Tweets no cargan"
  ├─ Widget → Flutter logs
  └─ Backend → logs de Spring Boot

Ver más en: SECURITY_GUIDE.md (última sección)
